Metodología Scrum

Scrum es un framework ágil diseñado para equipos que desarrollan productos complejos. Se basa en el control de procesos empírico, donde las decisiones se toman basadas en la experiencia y la observación.

01. Pilares

  • Transparencia: Visibilidad para todos.
  • Inspección: Chequeo constante del progreso.
  • Adaptación: Ajustar si algo sale mal.

02. Roles

Rol Misión
Product Owner Maximiza el valor.
Scrum Master Elimina impedimentos.
Developers Crean el incremento.

03. Eventos del Sprint

Planning → Daily → Review → Retrospective
